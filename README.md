# VibeStudio: AI-Native Web IDE

**"Where Code Meets Flow"** - A revolutionary browser-based IDE with AI at its core.

VibeStudio is the next generation of development tools, bringing VS Code functionality to your browser with cutting-edge AI capabilities and a beautiful, Windsurf-inspired interface. No downloads, no setup, just open and code.

## 🚀 Quick Start

```bash
# Clone and install
git clone <repo-url>
cd vibestudio
yarn install

# Start development  
yarn dev  # Opens localhost:3000
```

**Live Demo**: [app.vibestudio.online](https://app.vibestudio.online)

## ✨ Key Features

🌐 **Zero Setup** - Browser-native development, no downloads required  
🤖 **AI-First Design** - Built from the ground up with AI as core architecture  
💫 **Beautiful & Functional** - Windsurf-inspired UI with professional IDE power  
⚡ **Real-time Collaboration** - Live editing with intelligent conflict resolution  
🔥 **Firebase Backend** - Production-grade infrastructure with enterprise security

## 🔧 Tech Stack

**Frontend**: React 19 + TypeScript + Vite + Tailwind CSS  
**Backend**: Firebase (Auth, Firestore, Storage, Functions, Realtime)  
**Editor**: Monaco Editor (VS Code core)  
**AI**: OpenAI + Anthropic integration  
**State**: Zustand with persistence  
**Animations**: Framer Motion
